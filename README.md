## NeHe-GLFW3
Original NeHe OpenGL lessons, ported to GLFW3

## Compiling

MinGW *glfw3.a* static library is included in the project

- Get [Code::Blocks](http://www.codeblocks.org/downloads)
- Press **F9**
- Done!

Tested with GCC 7.2.0
